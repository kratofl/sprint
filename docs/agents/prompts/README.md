## Prompt Templates

These are reusable task prompts for agents working in this repository.

### Use These Prompt Templates For

- Large maintenance passes and documentation drift checks
- Bootstrapping new API endpoints
- Adding new shared UI components
- Adding new game adapters

### Available Prompts

- `docs/agents/prompts/maintenance.md`
- `docs/agents/prompts/new-api-endpoint.md`
- `docs/agents/prompts/new-shared-component.md`
- `docs/agents/prompts/new-game-adapter.md`

### Rule

Treat these as task recipes, not always-on policy. Always follow `AGENTS.md` and `docs/agents/*`
 first, then use a prompt template when the task matches.
