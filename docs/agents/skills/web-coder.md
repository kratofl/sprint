## Web Coder Skill

This is a generic helper skill for web work. It is not a source of truth for Sprint-specific
conventions.

### Use It For

- HTML, CSS, JavaScript, and browser API implementation
- accessibility and performance checks
- HTTP, caching, and web security discussions

### Constraint

Always defer to Sprint-specific repo guidance first:

- `AGENTS.md`
- `docs/agents/react-typescript.md`
- `docs/agents/nextjs.md`
- `docs/agents/tooling.md`

Use this skill only when general web standards knowledge is helpful beyond repo-specific rules.
